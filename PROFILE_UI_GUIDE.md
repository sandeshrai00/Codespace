## Color Scheme

- **Primary**: Gray-900 (#111827) - Main text, active states
- **Secondary**: Gray-700 (#374151) - Secondary text
- **Borders**: Gray-200 (#E5E7EB) - Card borders, dividers
- **Background**: Gray-50 (#F9FAFB) - Page background
- **White**: #FFFFFF - Card backgrounds
- **Success**: Green-50/800 - Success messages
- **Error**: Red-50/800 - Error messages

---

## Icons Used

- **Profile**: User circle icon
- **Edit/Profile Settings**: Edit/pencil icon
- **Account Settings**: Gear/settings icon
- **Arrow**: Right arrow for navigation cards
- **Country Flags**: Emoji flags in dropdown

---

## Interaction States

### Buttons
- **Default**: White background, gray border, gray text
- **Hover**: Light gray background
- **Active**: Gray-900 background, white text
- **Disabled**: Reduced opacity, cursor not-allowed

### Form Inputs
- **Default**: White background, gray border
- **Focus**: Ring (2px) gray-900, border transparent
- **Error**: Red border, red text
- **Disabled**: Reduced opacity

### Dropdown
- **Closed**: Shows selected country code
- **Open**: Full list with search
- **Hover**: Light gray background on items
- **Selected**: Indicated with checkmark or bold text

---

## Accessibility Features

1. **ARIA Labels**: All interactive elements have appropriate labels
2. **Keyboard Navigation**: Full keyboard support for all interactions
3. **Focus Indicators**: Visible focus states on all focusable elements
4. **Screen Reader Support**: Proper semantic HTML and ARIA roles
5. **Color Contrast**: WCAG AA compliant color combinations
6. **Error Messages**: Associated with form fields via aria-describedby
